## About The Project

![GitHub Logo](/app.png)


This Project shows us the weather of any city present in the globe, just enter the name of your/any city from around the world to get the exact weather condtions.

I have used HTML for basic structure, CSS for some text, image formatting, and made this responsive and attractive using CSS and its elements. Also used Javascript for making my website behave or work as I wanted, and linked the weather API which tells us the weather.

## Usage

This web app can provide the live temperature / climate conditions of any city present around the globe, it is also useful for tracking the climatic conditons.
